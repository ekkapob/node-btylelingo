Website Development 101
เริ่มต้นสร้างเว็บไซต์อย่างมือโปร

Web Development 101 มุ่งสู่การเป็น professional web developer โดยจะเน้นสอนตั้งแต่ การวางแผน software development ในทีมโดยใช้ Agile methodology, การออกแบบเว็บไซต์, การติดตั้งเครื่องมือที่จำเป็นในการสร้างเว็บไซต์, การใช้ html template, sass, jquery เพื่อช่วยในการสร้างเว็บไซต์ อย่างมืออาชีพ รวมถึงพื้นฐานการเขียน server-side programming และ database

หลังจากจบหลักสูตร นักเรียนจะมีเว็บไซต์ online ของตัวเองเพื่อนำไปต่อยอด และเป็น portfolio ในสายอาชีพต่อไป

เหมาะสำหรับนักพัฒนาโปรแกรม ที่มีความรู้เบื้องต้นในการเขียนโปรแกรมหรือเว็บไซต์ เข้าใจ HTML CSS Javascript Database

อธิบายรายละเอียด ขั้นตอนการทำงาน ตั้งแต่เริ่มต้น จนจบงาน

  • การใช้ Agile methodology ในการบริหาร software development
  • Story planning
  • Test-driven development (TDD)
  • Test automation*

วางระบบเพื่อพัฒนาเว็บไซต์ได้อย่างมีประสิทธิภาพ

  • ใช้เครื่องมือของ Node.js เพื่อพัฒนาเว็บไซต์ได้อย่างมืออาชีพและต่อเนื่อง
  • ใช้ source control เช่น Git
  • Gulp.js, Webpack, etc.

การบริหารจัดการ source code ในทีม

  • ใช้ source control เช่น git
  • การใช้ Git command ต่างๆ commit, merge, rebase, etc.

HTML5 / CSS3

  • ใช้ Bootstrap grid สำหรับสร้าง Responsive website
  • เครื่องมือต่างๆ ที่ช่วยสร้างเว็บไซต์ เช่น Sass, Handlerbars

Javascript อย่างมืออาชีพ

  • จัดการ HTML DOM, Events, Animations
  • AJAX
  • ครอบคลุมมาตรฐานใหม่ ES6
  • เครื่องมือที่ช่วยพัฒนา web app เช่น React*

การออกแบบเว็บไซต์

  • Sketch, Photoshop, Illustrator

ติดตั้งเว็บไซต์บน production

  • จดทะเบียน Domain name
  • ติดตั้งเว็บไซต์บน cloud platform เช่น Amazon, Digital Ocean, Heroku

การพัฒนา Backend programming และ Web services

  • Node.js
  • RESTful API
  • Ruby on Rails*

Database - Relational & NoSQL

  • CRUD Operations
  • PostgreSQL, MongoDB
